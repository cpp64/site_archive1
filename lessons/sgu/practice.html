1. Согласно обновленным ФГОС ООО составить КТП на 7,8,9 классы по предмету «Информатика». Нужны только темы, связанные с программированием, остальные темы заполнять не требуется. КТП на 28 часов за год (28 уроков).
  2. Зарегистрироваться на платформе codeforces.com. Прислать список логинов с ФИО к каждому логину.
  3. Войти в платформу polygon.codeforces.com
  4. В полигоне составить задачу A+B
    1. Входные данные — пара целых чисел
    2. Выходные — одно целое число
    3. Тестер — ncmp (сравнение целых чисел)
    4. Валидатор — не требуется
    5. Ограничения по времени — стандартные
    6. Тесты — 3 любых теста, без использования скриптов для создания тестов
    7. Условие — на русском языке, можно без использования Latex
  5. Т.к. codeforces для создания мэшапа (сборника задач) требует хотя бы 3 участия в рейтинговых раундах, необходимо:
    1. Зарегистрироваться на 3 ближайших рейтинговых раунда. Лучше всего Div.4 или Div.3 (чем больше номер Div, тем проще задачи).
    2. Отправить хотя бы 1 задачу в каждом раунде
  6. В Polygon создать по 5 задач на следующие темы:
    1. Простейшие арифм. операции, обычные задачи на логику (наподобие )
    2. Условный оператор
    3. Цикл while
    4. Цикл for

    Примечание: очевидно, все придуманные задачи для Полигона должны быть уникальными как у одного человека так и по всей группе.

  7. Добавить условия из созданных в Полигоне задач в HTML-документ и выложить в любой бесплатный онлайн-сервис (Github-pages или т.п.).
