Урок

--------------I вариант------------------

  1. При помощи цикла for вывести на экран числа от -12 до 15.
  2. Вход: количество чисел n массива и сами n чисел, а также два числа a и b. Выход: элементы массива с номерами начиная с a и заканчивая b.
  3. Ввести и вывести двумерный массив из n строк и m столбцов.

--------------II вариант------------------

  1. При помощи цикла for вывести все нечентные числа от -50 до 50.
  2. Вход: количество чисел n массива и сами n чисел, а также число a. Выход: количество элементов массива, которые кратны числу a.
  3. Ввести двумерный массив из n строк и m столбцов. Вывести 3-й столбец.

--------------III вариант------------------

  1. Вход: целое число n. При помощи цикла for вывести на экран сообщение "Good morning" ровно n раз.
  2. Вход: количество чисел n массива и сами n чисел, а также число a. Выход: количество элементов массива, которые равны числу a.
  3. Ввести двумерный массив из n строк и m столбцов. Вычесть из 2-го стобца 1-й. Вывести массив после операции.

--------------IV вариант------------------

  1. При помощи цикла for вывести на экран все числа, кратные 17 из интервала (0; 1000).
  2. Вход: количество чисел n массива и сами n чисел, а также число a. Выход: массив, в котором каждый элемент заменен на остаток от деления его на число a.
  3. Ввести двумерный массив из n строк и m столбцов. Найти кол-во четных элементов во 2 строке. Вывести результат.

--------------V вариант------------------

  1. При помощи цикла for найти сумму чисел от -40 до -10 и вывести на экран.
  2. Вход: количество чисел n массива и сами n чисел, а также два числа a и b. Выход: количество элементов массива, которые равны a+b.
  3. Ввести двумерный массив из n строк и m столбцов. Найти кол-во нулевых элементов в 1 строке. Вывести результат.

--------------VI вариант------------------

  1. Вход: целое число n. Выход: числа от 1 до n, не оканчивающиеся на 0.
  2. Вход: количество чисел n массива и сами n чисел, а также числа a и b. Выход: разность элементов, имеющих номера a и b соответственно.
  3. Ввести двумерный массив из n строк и m столбцов. Вывести 2-ю строку.

--------------VII вариант------------------

  1. При помощи цикла for вывести все чентные числа от -30 до 70.
  2. Вход: количество чисел n массива и сами n чисел, а также число a. Выход: массив, в котором каждый элемент заменен на частное от деления его на число a.
  3. Ввести двумерный массив из n строк и m столбцов. Обменять местами 4-й и 1 стобцы. Вывести полученный массив.

--------------VIII вариант------------------

  1. Вход: целое число n. Выход: числа от 1 до n, кратные 4.
  2. Вход: количество чисел n массива и сами n чисел, а также число a. Выход: количество элементов массива, которые не кратны числу a.
  3. Ввести двумерный массив из n строк и m столбцов. Прибавить к 1-му стобцу 3-й. Вывести массив после операции.

--------------IX вариант------------------

  1. При помощи цикла for вывести на экран все числа, не кратные 11 из интервала (0; 1000).
  2. Вход: количество чисел n массива и сами n чисел, а также два числа a и b. Выход: количество элементов массива, которые не равны a-b
  3. Ввести двумерный массив из n строк и m столбцов. Вывести сумму элементов 2-го столбца.

--------------X вариант------------------

  1. Вход: целое число n. При помощи цикла for вывести на экран сообщение "Good evening" ровно n раз.
  2. Вход: количество чисел n массива и сами n чисел, а также число a. Выход: количество элементов массива, которые не равны числу a.
  3. Ввести двумерный массив из n строк и m столбцов. Найти кол-во нечетных элементов в 3 строке. Вывести результат.

--------------XI вариант------------------

  1. Вход: целое число n. Выход: числа от 1 до n, оканчивающиеся на 0.
  2. Вход: количество чисел n массива и сами n чисел, а также два числа a и b. Выход: количество элементов массива, которые больше a, но меньше b.
  3. Ввести двумерный массив из n строк и m столбцов. Найти кол-во ненулевых элементов в 1 строке. Вывести результат.

--------------XII вариант------------------

  1. Вход: целое число n. Выход: числа от 1 до n в порядке убывания. (Например при n = 5: 5 4 3 2 1)
  2. Вход: количество чисел n массива и сами n чисел, а также числа a и b. Выход: сумма элементов, имеющих номера a и b соответственно.
  3. Ввести двумерный массив из n строк и m столбцов. Вывести сумму элементов 3-й строки.
