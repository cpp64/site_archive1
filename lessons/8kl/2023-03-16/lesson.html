Выполните работу либо в Pixel либо в онлайн-платформе Khan Academy

--------------I вариант------------------

  1. При помощи любого цикла постройте фигуру
  2. Ввести двумерный массив из n строк и m столбцов. Заменить все отрицательные числа на нули. Вывести массив после замен.
  3. Ввести двумерный массив из n строк и m столбцов. Заменить первую строку на вторую. Вывести массив после замены.

--------------II вариант------------------

  3. Ввести двумерный массив из n строк и m столбцов. Прибавить к первой строке вторую. Вывести массив после операции.
